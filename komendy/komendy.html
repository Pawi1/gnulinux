 - sudo  Wykonuje zadanie z uprawnieniami roota (odpowiednik administratora w windowsie)

Przykład:
  $ sudo apt update
Operacja aktualizuje liste reposytoriów i ich pakietów (distro oparte na debianie i pochodnych)

Przykład 2:
  $ sudo bash
  #
Uruchamia bash (odpowiednik cmd w linuxie) z uprawnieniami administratora (będziesz zalogowany jako root ($ = zwykły użytkownik # = root))

Przykład:
  $ man cat

Działania na plikach:



 - pwd  Wyświetla ścieżke naszego katalogu w jakim aktualnie się znajdujemy (od 'print working directory')

Przykład:
  $ pwd
  /home/user


 - cd  Komenda umożliwia zmiane katalogu (od 'change directory')

Przykład:
  $ cd [ścieżka katalogua albo nazwa katalogu]
  $ cd UlubioneFilmy
  $ cd /mnt/USB/Katalog


 - ls  Wypisanie informacji o PLIKACH (domyślnie w katalogu bieżącym). Sortowane alfabetyczne, jeżeli nie jest podana żadna z opcji (odpowiednik dir)

Przykład
  $ ls
   Katalog1 Katalog2 X Program.run
Kolory: niebieski = katalog, zielony = plik wykonywalny (executable) itd.
Więcej informacji uzyskarz dzięki --help lub man
